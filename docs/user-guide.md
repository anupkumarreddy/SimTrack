# User Guide

## Create Project

Staff users can create projects from the Projects page. Regular users can view project information but cannot create, edit, or delete records.

## Create Regression

Staff users can add regressions under a project. A regression is the stable definition for a test suite or workflow.

## Add or Import Run

Staff users can create runs from the Runs page. Each run stores branch, suite, configuration, counters, and result rows.

## View Results

Open a run or regression detail page to inspect pass rate, fail rate, individual results, and failed tests.

## Group Failures

Failure signatures group related failed tests for a run. Use the failure signature pages to understand recurring failures and affected tests.

## Interpret Dashboard

The dashboard shows:

- total projects, regressions, runs, and milestones
- project-level pass rate
- milestone completion progress
- highest failing and highest passing regressions
- upcoming milestones
- top failure signatures by tests affected
